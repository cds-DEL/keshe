## 2 系统总体方案设计
### 2.1 系统设计总体概述
本项目基于STM32F103C8T6主控芯片，以FreeRTOS实时操作系统为核心，搭建多功能嵌入式智能手表系统。系统集成电源管理、OLED显示、姿态传感采集、编码器人机交互、蓝牙数据通信、运动计步六大核心功能，采用面包板搭建硬件电路，无PCB设计。通过多任务实时调度实现界面刷新、传感器采样、数据运算、蓝牙交互并行运行，最终实现时间显示、姿态检测、多级菜单操作、运动计步记录、手机蓝牙数据同步等完整功能，满足课程设计基础要求及多项扩展要求。

### 2.2 系统四层总体架构
1. 硬件层：包含主控最小系统、电源模块、OLED显示模块、MPU6050传感模块、EC11编码器模块、HC-05蓝牙模块，为系统提供硬件运行基础。
2. 驱动层：基于STM32标准库开发I2C驱动、串口驱动、定时器驱动、编码器中断驱动，实现底层硬件的读写与控制。
3. RTOS系统层：移植FreeRTOS内核，完成任务划分、优先级配置、信号量与消息队列通信配置、SysTick时钟节拍配置，实现多任务实时调度。
4. 应用功能层：实现时间计时、OLED界面显示、姿态数据解析、多级菜单交互、计步算法运算、运动数据存储、蓝牙数据同步等上层业务功能。

## 3 硬件详细设计
### 3.1 核心主控电路设计
主控采用STM32F103C8T6最小系统板，板载复位电路、8MHz外部晶振、BOOT启动配置电路，无需自行绘制核心电路。全板外设统一由HT7333输出3.3V供电，匹配MCU额定工作电压。

### 3.2 电源电路设计
电源系统分为充电电路与稳压供电电路两部分：
- 充电电路：TP4056 Type-C一体化充电模块，支持5V外接输入为3.7V锂电池充电，内置过充、过放、短路保护。
- 稳压电路：锂电池输出电压范围3.0~4.2V，经过HT7333线性LDO稳压至稳定3.3V；串联1N4148二极管实现电源防反接，输入输出端并联100nF滤波电容，降低电压纹波。

### 3.3 外设接口电路设计
#### 3.3.1 I2C外设电路（OLED+MPU6050）
主控硬件I2C1引脚：PB6(SCL)、PB7(SDA)，两条总线各外接10kΩ上拉电阻至3.3V。OLED与MPU6050挂载同一组I2C总线，依靠不同设备地址区分通信对象，节省IO资源，布线简洁。

#### 3.3.2 EC11旋转编码器电路
编码器A、B两相引脚分别接入PA0、PA1外部中断引脚，中间按键引脚接入PA2普通GPIO；编码器公共端接GND，IO口内部上拉，双边沿中断检测旋转动作，实现菜单切换、数值调节，短按确认菜单选项。

#### 3.3.3 HC-05蓝牙串口电路
HC-05模块TX接STM32 USART1 RX(PA10)，HC-05 RX接STM32 USART1 TX(PA9)；模块VCC接3.3V，GND共地，波特率固定9600，8N1串口模式，实现双向透传数据交互。

### 3.4 关键元件参数计算
1. 稳压参数：HT7333输入耐压12V，压差仅0.3V，锂电池3.0~4.2V输入可稳定输出3.3V，满足所有外设工作电压。
2. I2C上拉电阻：10kΩ匹配400kHz高速I2C时序，总线电平稳定，无通信丢包。
3. 整机功耗估算：系统静态运行电流约20mA，150mAh锂电池理论续航7.5小时，完全满足实验室调试需求。

### 3.5 硬件布局规划（面包板无PCB）
1. STM32最小系统板放置面包板中间区域，缩短所有外设接线长度，减少信号干扰；
2. 电源模块、锂电池统一布置在面包板最左侧，远离I2C、串口信号线；
3. OLED显示屏、EC11编码器布置外侧，方便实时观察界面与手动操作；
4. MPU6050、HC-05蓝牙分区放置，电源线与信号排线分开走线；
5. 模块分区清晰，接线有序，便于后期故障排查、线路整改。

## 4 软件详细设计
### 4.1 软件整体模块划分
系统软件分层模块化开发，七大核心模块相互解耦：
底层驱动模块、系统计时模块、FreeRTOS任务调度模块、OLED显示UI模块、传感器采集&计步模块、编码器菜单交互模块、蓝牙通信模块。

### 4.2 FreeRTOS任务划分与调度设计
系统采用抢占式调度，SysTick配置1ms系统节拍，共创建6个独立任务，参数如下：

| 任务名称 | 优先级 | 堆栈大小 | 任务功能 | 运行机制 |
|--------|-------|---------|---------|---------|
| 时间更新任务 | 4（最高） | 128 | SysTick累加计时，更新时分秒全局变量 | 1s周期阻塞运行 |
| 传感器采集任务 | 3 | 256 | 定时读取MPU6050三轴加速度、角速度 | 10ms定时采集 |
| 计步运算任务 | 3 | 256 | 数据滤波、峰值检测、步数累加、本地存储 | 传感器数据队列触发运行 |
| OLED显示任务 | 2 | 128 | 渲染时间页、传感器数据页、多级菜单页面 | 20ms周期刷新 |
| 菜单交互任务 | 2 | 128 | 检测编码器旋转、按键事件，切换菜单层级 | 事件阻塞，无操作休眠 |
| 蓝牙通信任务 | 1（最低） | 256 | 接收手机指令、上传计步/时间数据 | 串口中断唤醒处理 |

### 4.3 任务间通信与同步设计
1. 二值信号量：I2C总线互斥信号量、OLED屏幕互斥信号量，防止多任务同时占用硬件造成通信错乱、花屏。
2. 消息队列：传感器数据队列、蓝牙收发数据队列，实现任务异步数据传输，降低耦合。
3. 阻塞休眠机制：无数据、无操作时任务进入阻塞态，释放CPU资源，降低系统整体功耗。

### 4.4 核心计步算法描述（扩展4核心）
采用滑动窗口均值滤波+三轴合成加速度峰值阈值算法：
1. 预处理：对MPU6050原始三轴加速度做5阶滑动平均滤波，滤除手部高频抖动噪声；
2. 合成加速度计算：sqrt(Ax²+Ay²+Az²)，消除手表佩戴倾斜带来的数值偏移；
3. 峰值检测：识别合成加速度连续“波峰-波谷”周期性变化；
4. 防抖阈值：设置300ms最小步间隔，过滤小幅抬手、晃动误计数；
5. 数据存储：累计当日总步数，变量缓存历史步数，支持蓝牙指令清零、数据上传手机上位机。

### 4.5 外设配置方案
1. I2C1：时钟400kHz高速模式，PB6/PB7复用I2C功能；
2. USART1：波特率9600，数据位8，停止位1，无校验，开启接收中断；
3. SysTick：1ms中断，作为FreeRTOS系统时钟节拍与软件计时基准；
4. 外部中断EXTI0/EXTI1：编码器A/B双边沿触发，上升下降沿均可检测旋转方向；
5. 低功耗配置：空闲任务开启STOP低功耗模式，闲置外设时钟关闭，降低功耗。

## 附：核心框架示例代码
```c
/* FreeRTOS任务创建核心代码片段 main.c */
#include "stm32f10x.h"
#include "FreeRTOS.h"
#include "task.h"
#include "queue.h"
#include "semphr.h"
#include "oled.h"
#include "mpu6050.h"
#include "encoder.h"
#include "bluetooth.h"

/* 全局句柄定义 */
SemaphoreHandle_t xI2CSemaphore;
QueueHandle_t xSensorDataQueue;

/* 任务函数声明 */
void vTimeTask(void *pvParameters);
void vSensorTask(void *pvParameters);
void vStepCountTask(void *pvParameters);
void vOledDispTask(void *pvParameters);
void vMenuTask(void *pvParameters);
void vBtCommTask(void *pvParameters);

int main(void)
{
    //硬件底层初始化
    STM32_Init();
    OLED_Init();
    MPU6050_Init();
    Encoder_Init();
    Bluetooth_Init();

    //RTOS资源创建
    xI2CSemaphore = xSemaphoreCreateBinary();
    xSemaphoreGive(xI2CSemaphore);
    xSensorDataQueue = xQueueCreate(16, sizeof(MPU6050_Data));

    //创建6个业务任务
    xTaskCreate(vTimeTask, "TimeTask", 128, NULL, 4, NULL);
    xTaskCreate(vSensorTask, "SenTask", 256, NULL, 3, NULL);
    xTaskCreate(vStepCountTask, "StepTask", 256, NULL, 3, NULL);
    xTaskCreate(vOledDispTask, "OledTask", 128, NULL, 2, NULL);
    xTaskCreate(vMenuTask, "MenuTask", 128, NULL, 2, NULL);
    xTaskCreate(vBtCommTask, "BtTask", 256, NULL, 1, NULL);

    vTaskStartScheduler();
    while(1);
}

//计步算法核心片段
void Step_Calc(MPU6050_Data data)
{
    float acc = sqrtf(data.ax*data.ax + data.ay*data.ay + data.az*data.az);
    static float buf[5];
    static uint8_t idx = 0;
    static uint32_t step_time = 0;
    static uint32_t step_cnt = 0;

    //滑动平均滤波
    buf[idx++] = acc;
    if(idx >= 5) idx = 0;
    float avg = (buf[0]+buf[1]+buf[2]+buf[3]+buf[4])/5.0f;

    //峰值阈值判断+防抖间隔
    if(avg > 1.2f && (millis()-step_time)>300)
    {
        step_cnt++;
        step_time = millis();
    }
}
```
